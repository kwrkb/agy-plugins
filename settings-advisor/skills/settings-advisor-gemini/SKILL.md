# Settings Advisor スキル

このスキルは、カレントワークスペースのコードベース規模や言語、タスクの性質に応じて、agy (Antigravity) の設定パラメータ（モデル、サンドボックス、パーミッションなど）の最適な構成を提案します。

## 使用方法

ユーザーから「設定の推奨値は？」「どのモデルを使うべき？」「セキュリティ設定はどうすべき？」といった質問があった場合、または大規模なタスクを開始する前に、`settings-advisor` MCP ツールを呼び出してください。

```
settings_advisor(path="<リポジトリパス>", task_hint="<タスクの概要や指示内容>")
```

## 出力の扱い

ツールから返された提案（モデルの推奨、サンドボックスの有効化など）をそのままユーザーに提示し、`/model` コマンドや設定パネル（`/settings`）から手動で適用できることを案内してください。
